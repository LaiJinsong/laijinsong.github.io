Selected Projects

Master Thesis — Multimodal Spatial Arithmetic with Tangibles

A multimodal learning and interaction system that supports spreadsheet-style arithmetic for blind and visually impaired users through a tactile grid, Braille-labeled tangibles, speech interaction, and camera-based tracking.

My role: I designed and implemented the prototype end-to-end, including the tangible interaction concept, computer-vision modules (grid detection, tangible recognition, hand tracking), arithmetic logic, speech input/output integration, and an initial user evaluation for usability and learning support.

Tech stack: Python · MediaPipe · OpenCV · Raspberry Pi · ASR · Piper TTS · Multimodal Interaction

MultiSense Dashboard — Accessible Real-Time Monitoring

A multimodal dashboard for blind and visually impaired users to monitor real-time sensor data via speech and sonification, with Kafka-based streaming and screen-reader-compatible interaction.

My role: I was mainly responsible for frontend page development, page configuration logic, and dashboard display logic, including UI implementation, configuration workflows, and real-time data integration with backend services.

Tech stack: Vue.js · Spring Boot · Apache Kafka · WebSocket (STOMP/SockJS) · Web Audio API · SpeechSynthesis · ARIA

Links: Repository Demo + Slides

Smart ANC — Adaptive Noise Masking

A smart headphone support system for autistic users in office environments, combining ANC control, ambient sound monitoring, and adaptive masking noise to reduce distracting sounds such as speech and background noise.

My role: I was mainly responsible for the ambient noise masking module, including masking strategy research and real-time masking-noise generation/adaptation based on detected sound types (for example voice vs. non-voice), with a focus on smooth transitions and reduced distraction.

Tech stack: Python · Django · SQLite · WebSocket · OpenAPI · PyTorch · Silero VAD/STT · NumPy · torchaudio
